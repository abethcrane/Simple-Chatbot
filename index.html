Spreadsheet Editing Chatbot

It's not a huge project, and the chatbot uses a regex on each line (defined as until a \n) to determine what functions to call.

Set Up

Edit the following fields, and then just run the script ./chatbot.py for an email you have as a contact.

  • Email
  • Password
  • Spreadsheet Key
  • Admin email

Authors and Contributors

In early 2013 Beth Crane (@bethc) wrote this bot, with the help of a few code samples and libraries online.
